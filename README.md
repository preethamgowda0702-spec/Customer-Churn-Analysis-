

📊 Customer Churn Analysis and Prediction

🚀 Project Overview

This project analyzes customer churn data and predicts whether a customer is likely to leave a service using Machine Learning.

The project combines:

- 📈 Data Analysis
- 📊 Data Visualization
- 🤖 Machine Learning
- 🎯 Customer Insights
- 🌐 Interactive Streamlit Dashboard

to help businesses understand customer behavior and improve retention strategies.


Dataset Columns


- CustomerID
- Age
- Gender
- Tenure
- Usage Frequency
- Support Calls
- Payment Delay

- Churn
